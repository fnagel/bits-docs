# bind

▸ bind(mount, bit): Promise<void>

Used to bind this instance as a glue layer between the DOM nodes and the reactive data of the given bit. It will also bind statically declared event listeners (used by the decorator) to the DOM

This is a internal process that MUST be executed ONLY ONCE! Multiple binds on the same bit will cause the universe to grind to a halt; and we don't want that to happen, right?

# Parameters

Name Type Description
mount Mount The dom node of the bit mount
bit AbstractBit The instance of the bit class

# Returns

Promise<void>

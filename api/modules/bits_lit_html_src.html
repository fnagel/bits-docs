# dataModel

▸ Const dataModel(property): (part: any) => Promise<void>

Directive to simplify two-way data binding on form elements.

You simply bind it to the input event it should listen for (change/keydown for the most part) and tell it which property you want to bind it to.

The binding will then update the input value in the same way data-model would.

You can also use @context if you would really want to.
